# 🎓 Student Management System

A Python and SQLite based CRUD application for managing student records efficiently.

## Features

✅ Add Student  
✅ View Students  
✅ Update Student  
✅ Delete Student  
✅ Search Student  

## Technologies Used

- Python
- SQLite
- SQL



## How to Run

```bash
python main.py
```
