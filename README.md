# 🎓 Student Management System

A Python + SQLite based CRUD application for managing student records.

## Features

- Add Student
- View Students
- Update Student
- Delete Student
- Search Student
- SQLite Database Integration

## Technologies Used

- Python
- SQLite
- SQL
- Git
- GitHub

## How to Run

```bash
python main.py
```
